USAGE EXAMPLES
───────────────────────────────────────────────────────────────────────

Quick Cycle (5 seconds):
  1. Open blink_test.spin2
  2. Edit code
  3. Press Ctrl+S
  4. Click "Compile & Run"
  → Program runs on board immediately!

Traditional Workflow:
  1. Click "Open" → Select SPIN2 file
  2. Edit code (see syntax highlighting)
  3. Click "Compile" → Check output
  4. Click "Load to RAM" → Test on board
  5. Click "Load to FLASH" → Store if working

Recovery:
  1. Program hangs
  2. Click "Reset Target"
  3. Edit code
  4. Click "Compile & Run" to try again
